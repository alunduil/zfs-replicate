# Getting Started

Welcome to the project. This document helps you set up your
development environment. Follow these steps to get started
quickly and efficiently.

## Prerequisites

Make sure you have Python 3.10 or later installed on your
system. We recommend using Nix for reproducible development
environments, but Poetry also works well for most use cases.

## Installation Steps

Run `poetry install` to set up the project dependencies and
configure your local development environment.
